## 📱 Access Your Local Application

Once all services are running locally, you can access the different layers of the ecosystem using the links below:

| Component | Access Link / Command | Description |
| :--- | :--- | :--- |
| **Web Dashboard** | [http://localhost:5173](http://localhost:5173) | React/Vite Customer & Admin frontend |
| **Backend API Gateway** | [http://localhost:3000/api/v1](http://localhost:3000/api/v1) | Express REST API base endpoint |
| **Health Check** | [http://localhost:3000/health](http://localhost:3000/health) | API uptime and database connection status |
| **Local Database** | `psql -U localbites_user -d localbites` | Direct CLI connection to PostgreSQL |
